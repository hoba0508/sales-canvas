Sales Canvas

A dedicated sales command centre for your field team — customers, invoices, credit limits, visit scheduling, and approval workflow, all in one place.


Odoo 19 Sales Field Force Indonesia Ready

Key Features

🏪

Customer 360° View

Separate "Sales Canvas" menu that shows each customer's contact info, NPWP, Is PKP flag, credit limit, salesman, and real-time stat buttons — without touching the standard Contact form.

💰

Smart Stat Buttons

Instant totals for Sales Orders, Invoiced, Unpaid (Belum Lunas), Overdue, and remaining Credit. Overdue and over-limit amounts are highlighted in red.

📋

Inline Sales & Invoice Tables

Colour-coded inline tables for Sales Orders and Invoices directly on the customer card. Invoice list also shows Aged AR (days overdue) column.

🗺️

Salesman Activity & Visit Map

Full visit planning module (sc.visit): create planned visits, check-in with photo, checkout, and visualise the day's route on an interactive Mapbox map.

✅

Visit Approval Workflow

4-stage approval flow: Planned → Upload Photo → Awaiting Approval → Approved by Manager → Completed. A dedicated Manager Approval menu lists all pending visits.

🔒

Role-Based Access

Separate security groups for Salesman (field data entry) and Sales Manager (approval, full view). Easy to assign via Odoo's standard Users & Groups.

Installation & Configuration

Dependencies

  • contacts — Odoo Contacts app
  • sale_management — Sales Management
  • account — Accounting / Invoicing

Mapbox API Key (for Visit Map)

The interactive route map requires a free Mapbox token.
Set it in Settings → Technical → Parameters → System Parameters with key sales_canvas.mapbox_token.

⚠️ Note: This module does not modify the standard Odoo Contact form. All Sales Canvas views are served through a dedicated top-level menu, keeping your standard CRM and Contact views completely untouched.
